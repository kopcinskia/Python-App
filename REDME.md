# Python App

## Start:
### Watch JS to develop
```
$ cd fullstack_template/static/
$ npm install
$ npm run watch
```

### Create virtualenv:
```
$ virtualenv venv
$ source venv/bin/activate
$ pip install flask
$ pip install -r requirements.txt
```

### Run App:
```
cd app/static/
$ npm install
$ npm run watch

cd app/server/
$ Python server.py
```
